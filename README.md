# Sass Unpack

Sass Unpack cut a sass project in multiple css files

## Install

Install with [npm](https://npmjs.org/package/sass-unpack)

```
npm install --save-dev sass-unpack
```

## Usage

Usage as a Node library:

```js
var sassUnpack = require('sass-unpack');
```
